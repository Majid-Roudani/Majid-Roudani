## Projets Phares

### [art700-legal-prediction - NLP Juridique](https://github.com/Majid-Roudani/art700-legal-prediction)
* **Défi :** Prédire l'application de l'**article 700 du Code de procédure civile** (frais de justice) à partir de décisions juridiques brutes.
* **Technique :** Pipeline complet TF-IDF + Voting Hard (ML), fine-tuning **CamemBERT** (LLM), **RAG** avec recherche vectorielle FAISS.
* **Stack :** Python, Scikit-learn, LightGBM, PyTorch, Transformers, FAISS, Streamlit.
* 🔗 [Voir l'application en ligne](https://art700-legal-prediction.streamlit.app/)

### [assurance-auto-analyse - Analyse prédictive assurance](https://github.com/Majid-Roudani/assurance-auto-analyse)
* **Défi :** Prédire la durée de traitement d'un dossier d'assistance automobile et anticiper l'attribution d'un véhicule de remplacement, sur **100 000 dossiers**.
* **Technique :** Modélisation économétrique (OLS, Logit) et Machine Learning (Gradient Boosting, XGBoost, Random Forest).
* **Stack :** Python, statsmodels, Scikit-learn, XGBoost, Streamlit.
* 🔗 [Voir l'application en ligne](https://assurance-auto-analyse.streamlit.app/)
