## 🔬 Projets phares

### ⚖️ [art700-legal-prediction](https://github.com/Majid-Roudani/art700-legal-prediction)
Classification NLP/ML/DL de décisions de justice pour prédire l'application de l'**article 700 du Code de procédure civile** (frais de justice). Pipeline complet : TF-IDF + Voting Hard (ML), fine-tuning CamemBERT (LLM), RAG avec recherche vectorielle FAISS, déployé via Streamlit.

🔗 **[Voir l'application en ligne](https://art700-legal-prediction.streamlit.app/)**

### 🚗 [assurance-auto-analyse](https://github.com/Majid-Roudani/assurance-auto-analyse)
Analyse de 100 000 dossiers d'assistance automobile pour prédire la durée de traitement et anticiper l'attribution d'un véhicule de remplacement. Modélisation économétrique (OLS, Logit) et Machine Learning (Gradient Boosting, XGBoost, Random Forest), déployé via Streamlit.
